Project Phoenix

{% endblock %} {% block content %}

Plaid Hat Games ceased development of Ashes in April 2019. But the Phoenix will not be denied. Project Phoenix is a group of die-hard Ashes fans working together to continue development on this wonderful game.

Current projects

  • Time dice: Plaid Hat teased a seventh and final dice type on their deluxe expansions, but the game was canceled before Time dice were released. Developing and play-testing a custom implementation of Time dice is one of the main initial goals for Project Phoenix.
  • Time expansions: There are currently two expansions being designed and play-tested using the new Time dice (a mono-Time deck, and a Time/Charm deck).
  • New rulings: The official FAQ includes several counter-intuitive rulings, and sadly failed to define a small number of low-level game mechanics. Project Phoenix is play-testing a revised, more comprehensive ruleset (while maintaining compatibility with the core rulebook).
  • Bringing Ashes 500 up to date: The Ashes 500 alternate deck-building rules are alive and kicking, and Project Phoenix is updating them to cover all released Ashes cards.

Project Phoenix is still in its infancy, so we do not have a specific timeline we can share; however, the above projects have all been under active development even before Plaid Hat announced the official cancellation of Ashes. We hope to keep Ashes fresh for its dedicated fans for a long time to come.

Get involved

Development for new cards is currently being handled via the AshesChat Slack channel. If you're interested in helping design or test cards, please come say hi!

The work-in-progress Raven Rules are available here; please try them out and leave a comment about what worked and what didn't!

The most recent Ashes 500 values are live, so feel free to build some decks and post if something seems valued poorly!
